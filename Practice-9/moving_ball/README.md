# Moving Ball Game

This program displays a red ball on a white background.

## Features
- Ball size: 50x50 pixels
- Radius: 25
- Moves with arrow keys
- Each key press moves the ball by 20 pixels
- Ball cannot leave screen boundaries

## Controls
- Up Arrow = move up
- Down Arrow = move down
- Left Arrow = move left
- Right Arrow = move right

## Run
```bash
python main.py
