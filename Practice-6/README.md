# Practice 6

This project demonstrates Python exercises related to file handling, directory management, and built-in functions.

## Folder Structure

Practice6/

│   ├── read_files.py

## Tasks Implemented

### File Handling
- Create and write to text files
- Read and print file contents
- Append new lines to files
- Copy and backup files using shutil
- Safely delete files

### Directory Management
- Create nested directories
- List files and folders
- Find files by extension
- Move and copy files

### Built-in Functions
- map()
- filter()
- reduce()
- enumerate()
- zip()
- Type checking and conversions
